# JSON Validation Check

Checks that all `.json` files can be parsed.

## Supported Features

- ✅ Files
- ✅ Git Pre-Commit
- ✅ GitHub Actions
- ✅ Fixes

## Configuration

- `severity`: The severity of this check.

### Example Configuration

```json
{
  "ValidJson": {
    "severity": "ERROR"
  }
}
```
